h1. lib-deamon-cpp

Version: 1.0.1
Release Date: 2020-04-15

h2. Summary

Library to make daemons in c++ projects.

h2. Getting Started

h3. Build

pre. git clone https://github.com/PawelJanik/lib-deamon-cpp.git
cd lib-daemon-cpp
mkdir build
cd build
cmake ..
make

h3. Install

pre. make install (as root user)
ldconfig (as root user)
