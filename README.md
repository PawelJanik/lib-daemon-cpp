**lib-deamon-cpp**

**Version:** 1.0.1

**Release Date:** 2020-04-15

Library to make daemons in c++ projects.

**Build**
```
git clone https://github.com/PawelJanik/lib-daemon-cpp.git
cd lib-daemon-cpp
mkdir build
cd build
cmake ..
make
```

**Install**
```
make install (as root user)
ldconfig (as root user)
```
